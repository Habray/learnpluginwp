== Installation ==

= AUTOMATIC INSTALLATION (EASIEST WAY) =

To do an automatic install the plugin, log in to your WordPress dashboard, navigate to the Plugins menu and
click Add New. In the search field type "WordPress Post Slider". Once you have found it you can install it by simply
clicking "Install Now" and then "Activate".

= MANUAL INSTALLATION =

* Download the plugin
* Navigate to the ‘Add New’ in the plugins dashboard
* Navigate to the ‘Upload’ area
* Select the plugin from your computer
* Click ‘Install Now’
* Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= What is a carousel slider? =
It is a slideshow view of texts, photos, and videos in wordpress. The carousel slider adds visuals to a website’s texts, images, and videos by automatically displaying them at a certain motion, speed, and time. It helps get visitors’ attention and enhances the user experience.

= How do I add text to carousel in WordPress? =
You just need to add your post into the source box just below the Query option, make some changes in the setting, save the changes, copy the shortcode, and paste it into your specific page.

= How do I use carousel slider in WordPress? =
Once installing this plugin, go to the WordPress slider from your WP dashboard, make some changes in the settings if you want, and click on publish. Then you’ll get a shortcode. Simply copy the code and paste it into the page you wanted to add the carousel. Then publish it, and the carousel will be added to your page.

= How do you add text to Carousel Elementor? =
If you’re using Elementor, then first you need to go to your WordPress site, select the post, copy the shortcode just like before and click on edit with Elementor. Then from the Elementor dashboard, go to the element option from the left sidebar and search shortcode. After that, simply drag and drop the shortcode button and paste your shortcode into the box. And it’s done.   

= Can you put text over an image in WordPress? =
Yes, with the WordPress Carousel Slider With Text plugin, you can put text over an image in  Wordpress.

= What types of slider you can create using this plugin? =
You can create post sliders with images and other content of the posts.

= Do I need to have coding skills to use this plugin? =
No. You don’t need any coding skill. No more pain writing any shortcode attributes to display slider in your site. With just a few clicks, you can amazingly create unlimited slider.

= Are the sliders of the plugin responsive? =
Yes, they are fully responsive, mobile and touch-friendly.

= Can I add sliders anywhere on my website?=
Yes, you can add sliders anywhere else on your WordPress website with a shortcode or PHP function.

= Can I have multiple sliders with the separate setting?
Yes. You can have an unlimited slider with each separate setting in the page, post, template, widget or anywhere you like.

= Will the sliders work with my theme? =
Yes, they work with any WordPress theme if themes are developed according to WordPress standard.
